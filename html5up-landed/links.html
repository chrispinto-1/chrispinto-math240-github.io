Helpful Links

Below are websites that will be extremely useful for this course. Feel free to use them or not! Make sure to always check the class canvas page frequently.

For Class

Prof D will send out the zoom link so check your emails!

For Graphing

Desmos

Desmos is a great tool for 2D plots of any function involving real numbers. You can find intersections of curves, plot points and shapes, show time evolving functions, etc.

GeoGebra

GeoGebra is similar to Desmos but for 3D graphs. I personally only use it for visualization purposes, and not so much computational.

For Computations

Symbolab

Symbolab is great for problems you'd find in Calc I and II, but also has some matrix capabilites and ODE solvers.

Derivatives Integrals

I still too struggle with some derivatives and integrals! Whenever you appraoch a hard integral or are just lazy (it happens to all of us), use these tools.

Wolfram Alpha

Wolfram Alpha will come in handy when we start doing ODEs. Good way to check your work.

MatLab

Not necessary but MatLab is a useful tool for matrix operations, ODE solvers, and graphing. MatLab is specifically designed to handle matrices and linear algebra operations, so it is a great tool with a ton of features. Most UPenn students get a free license as well!
